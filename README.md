## How to use it with Google Colab?
Fetch the module: 
```
$ pip install git+https://github.com/clappy203/Calculator
```

Install Pytest with:
```
$ !pip install pytest ipython_pytest
```

Import package: 
```
$ from calculator_package.calculator import Calculator
```

Innitialize class: 
```
$ cal = Calculator()
```

Use features

```
$ cal.add(a)
```

```
$ cal.subtract(a)
```

```
$ cal.multiply(b)
```

```
$ cal.devide(b)
```

```
$ cal.nth_root(b)
```

```
$ cal.reset()
```
